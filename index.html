RIVER CROSSING GAME

No. of Crossing:

Rules

1. Transport all the members from the left bank to the right bank of the river

2. You can only use boat to transfer the members.

3. Boat have maximum capacity of 2 people. Boat can run only if there is someone present in the boat.

4. The number of zombies should never exceed the number of humans at either bank.

5. If the zombies are greater in number player will lose the game.
